# Skiddys Learning Platform

A modern learning platform built with React and PocketBase, designed to provide an interactive and engaging educational experience.

## Features

- Course Management System
- User Authentication
- Interactive Lessons
- Course Reviews
- Profile Management
- Privacy Settings
- Support System

## Tech Stack

- Frontend:
  - React
  - TypeScript
  - Tailwind CSS
  - React Router
  - React Query

- Backend:
  - PocketBase
  - SQLite

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn
- Git

### Installation

1. Clone the repository:
```bash
git clone [your-repository-url]
cd project
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Start the backend server:
```bash
cd backend
./pocketbase serve
```

## Deployment

The project can be deployed using:
- EasyPanel for container deployment
- Netlify for frontend hosting
- Fly.io for backend hosting
